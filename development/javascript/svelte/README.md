# Svelte

## Create app

Install npx.

```
sudo npm install -g npx
```

Create Svelte app.

```
npx degit sveltejs/template app
```

Change into directory of app and install npm packages.

```
cd /.app
npm install
```

Run app.

```
npm run dev
```

Go to local url shown, for me it was [http://localhost:5000](http://localhost:5000).
